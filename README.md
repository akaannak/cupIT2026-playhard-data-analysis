# Секция «Аналитика данных». Кейс от Procter & Gamble (P&G)


### Описание проекта

Проект выполнен в рамках кейса по аналитике данных и посвящён анализу присутствия бренда Procter & Gamble в генеративном поиске.

В работе исследуется, как нейросети формируют рекомендации товаров на основе пользовательских запросов и какие источники влияют на итоговый выбор пользователя.

**Основная задача** — выявить причины низкой представленности бренда в AI-ответах и предложить решения для повышения его видимости.

**Бизнес-задача**

Определить:

- почему продукты P&G редко попадают в ответы генеративного поиска
- какие источники влияют на рекомендации нейросетей
- как увеличить присутствие бренда в AI-выдаче
- какие точки роста можно использовать для увеличения трафика и продаж


### Данные

В рамках проекта использовались:

- поисковые запросы пользователей
- ответы генеративного поиска (Яндекс Нейро)
- источники, используемые ИИ при формировании ответов
- данные по конкурентам


### Ход работы

1. Анализ поисковой выдачи
   - сбор и структурирование поисковых срезов
   - анализ ответов генеративного поиска
   - выделение типов запросов (брендовые, категорийные, сравнительные и др.)
2. Анализ конкурентов

Проведено сравнение с основными конкурентами:
- Huggies (категория Baby)
- L’Oreal / ESTEL (Hair)
- R.O.C.S. / Philips (Oral)

Выявлено, что конкуренты стабильно присутствуют в источниках, используемых ИИ.

3. Анализ источников

Определены ключевые источники, влияющие на ответы нейросетей:
- маркетплейсы (в первую очередь Яндекс Маркет)
- профильные сайты
- экспертные и медицинские порталы

ИИ формирует ответы на основе доверенных источников, где P&G представлен слабо


### Ключевые выводы
- бренд P&G недостаточно представлен в источниках, используемых ИИ
- около 73% ответов по небрендовым запросам не содержат упоминаний P&G
- пользователи часто завершают поиск после AI-ответа, не переходя дальше
- отсутствие в AI-выдаче приводит к потере значительной доли потенциальных клиентов


### Разработка стратегии

Предложены решения:
- усиление присутствия на доверенных источниках
- публикации на профильных и экспертных площадках
- развитие присутствия на маркетплейсах
- работа с контентом под генеративный поиск

Также предложена стратегия продвижения через флагманские продукты (например, Pampers, Head & Shoulders, Oral-B).


### Оценка эффекта

Ожидаемые результаты внедрения:
- рост органического трафика на 40–60%
- увеличение узнаваемости бренда ~35%
- рост конверсии из небрендовых запросов ~25%


### Структура проекта
Ноутбук — анализ данных и запросов
Презентация — итоговые выводы и стратегия
Дополнительные материалы — графики и источники
